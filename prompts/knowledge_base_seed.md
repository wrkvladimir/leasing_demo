# Черновик структуры базы знаний для «Микро Лизинг»

Ниже не готовый фактический контент, а структура документов, которые стоит добавить в KB/RAG. Заполните их утвержденной информацией компании.

## Документы для оформления
- Документы для физического лица.
- Документы для юридического лица.
- Что делать, если документов нет под рукой.
- Куда загружать документы: личный кабинет / ссылка / менеджер.

## Процесс оформления
- Шаги после расчета графика.
- Срок рассмотрения заявки.
- Как клиент получает договор и график.
- Когда нужен менеджер.

## Ограничения и условия
- Доступные предметы лизинга по типу клиента.
- Что не финансируется.
- Валюты расчета.
- НДС: когда учитывается и как объяснять клиенту.
- Авансовый платеж: максимум 39%, минимум определяется калькулятором.

## Эскалация
- Когда переводить на менеджера.
- Какие данные спросить для обратной связи.
- Формулировка, если ответа нет в базе знаний.
